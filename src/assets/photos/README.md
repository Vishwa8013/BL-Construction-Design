Add your project media files in these folders:

- `src/assets/photos/house-design`
- `src/assets/photos/builders`
- `src/assets/photos/interior-design`
- `src/assets/photos/videos`

How it works:

- Images placed in `house-design` will appear under `House Design`
- Images placed in `builders` will appear under `Construction`
- Images placed in `interior-design` will appear under `Interior Design`
- Video files placed in `videos` will appear under `Videos`

Supported files:

- Images: `.png`, `.jpg`, `.jpeg`, `.webp`
- Videos: `.mp4`, `.mov`, `.webm`

File naming tip:

- Use clear names like `front-elevation.jpg` or `living-room-finish.mp4`
- The website converts the file name into the project title automatically
